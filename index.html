Kylo is a web browser with a large interface theme designed for computers connected to a TV. Kylo is intended to be used with an in-air pointer remote, such as the Logitech MX-Air or the Scoop™ from Hillcrest Labs. Kylo's default homepage is a curated collection of websites designed for lean-back TV experiences.
